# .impeccable.md — 岁吉设计语境

## Design Context

### Users
25-40岁都市人，对传统文化好奇但不迷信，习惯用好产品（Notion/小宇宙/Airbnb），对设计品质有感知力。使用场景：早上看看今日运势、睡前翻翻命盘、偶尔跟 AI 聊聊心事。

### Brand Personality
**温暖 · 小众 · 治愈**
- 不是算命 APP，是自我觉察工具
- 不冷冰冰，不说教，像一个懂你的朋友
- 有文化底蕴但不端着

### Aesthetic Direction

**参考 APP（提取设计基因）：**
- **Airbnb**：大圆角、照片驱动、clean layout、按钮有呼吸感
- **小宇宙**：温暖色调、触感按钮、微妙的拟物感、情感化设计细节
- **Notion**：信息密度适中、排版优雅、serif + sans 混排、克制的颜色
- **Teenager Engineer**：碰撞色、拟物质感、年轻但不幼稚、精致的工业美学

**风格定义：Neo-Tactile Warmth（新触感温暖）**
- 大圆角（16-24px），像手指触碰的柔软边缘
- 微妙的 elevation（不是生硬的 drop shadow，而是有层次的 inner shadow + soft outer）
- 暖色中性色底（不是冷灰白，是纸张色、奶茶色）
- 碰撞色点缀（朱砂红、青瓷蓝、鹅黄）——不是全页铺开，是一个小点让你注意到
- 拟物元素克制使用：日历像真的纸、按钮像真的可以按下去
- 字体：iOS 系统字体 SF Pro（正文）+ Noto Serif CJK（标题/大字）— 宋体的文化感 + SF Pro 的现代感混排

**反参考（绝对不要）：**
- ❌ 算命 APP 的红金配色、八卦图背景
- ❌ 冷色科技感（暗色+霓虹，太 tech）
- ❌ 到处都是虚线边框和 `fontWeight: '200'`（伪极简）
- ❌ 扁平得像纸片一样，没有层次和触感

### Design Principles

1. **触感优先**：每个可交互元素都要有物理感——按下去有回弹、有凹陷、有触觉反馈
2. **温度感**：色彩、圆角、动画都传递温暖，不冷不燥
3. **信息层次清晰**：不堆叠，用导航和展开来组织复杂信息
4. **iOS 原生感**：遵循 HIG，用标准导航模式，不重新发明轮子
5. **细节即品位**：每个过渡动画、每个间距、每个颜色过渡都要经得起放大看

### Technical Constraints
- React Native (Expo) — 不能用 CSS，只有 StyleSheet
- 优先 iOS 体验（App Store 首发）
- 系统字体为主（SF Pro），Noto Serif CJK 做补充
- react-native-reanimated 做动画
- 大圆角：radius 16 标准，12 紧凑，24 突出
