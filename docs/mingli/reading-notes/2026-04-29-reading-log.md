### `bazi-sanming-tonghui` — 《三命通会》四库全书本，卷二

- Source URL: https://zh.wikisource.org/wiki/三命通會_(四庫全書本)/卷02

  - `论六害`
  - `论三刑`

#### 关键摘录/事实

1. 人元司事的核心定义：
   - 原文大意：十干为天元，十二支为地元，支中所藏为人元；人元为司事之神，以命术言之为月令。
   - 产品/算法含义：月令不是只看地支名，而是看支中藏干在节气后不同天数的“当值”。

2. 人元司事表存在版本/体系差异，必须标为 contested：
   - 正文段落示例：寅中“艮土用事五日，丙火长生五日，甲木二十日”；卯中“甲木七日，乙木二十三日”。
   - 同页又引“玉井”表：正月寅“己土七日，丙火五日，甲木十八日”等。
   - 当前代码 `lib/bazi/SiLing.ts` 使用的是寅“戊7、丙7、甲16”、卯“甲10、乙20”等，与上述两组都不完全一致。
   - 结论：`bazi.siling.table` 不能直接标 high confidence；需要版本化表格，或至少在代码/claim 标明采用的是哪个传本/现代表。

3. 五行旺相休囚死：
   - 原文用“春木旺、火相、水休、金囚、土死”等说明五行随四时变化。
   - 重要提醒：原文也强调“生旺未必吉、休囚死绝未必凶”，“妙在识其通变”。
   - 产品含义：不能把五行数量/旺衰做成绝对好坏标签；应该表达为能量结构和需要调和的方向。

4. 六合/三合/六害/三刑：
   - 六合被解释为阴阳相和、律吕/日月会同等象数关系。
   - 三合强调“生、旺、库”成局，如申子辰水局、巳酉丑金局、寅午戌火局、亥卯未木局。
   - 六害、三刑不是简单“坏”，原文也按生旺/死绝/贵格/贱格讨论差异。
   - 产品含义：冲合刑害不应该直接输出恐吓式断语；应转成“关系张力/资源牵动/结构冲突”的语言。

#### 对当前 repo 的影响

- `lib/bazi/SiLing.ts` 的人元司令表需要拆成 claim，并标记 contested/versioned。
- `lib/bazi/BaziEngine.ts` 的旺衰、三合、六合、六害、三刑可以逐步回链到《三命通会》卷二，但解释层要保留“通变”原则，避免机械吉凶。

### `ziwei-doushu-quanshu` — 《紫微斗数全书》，卷一/卷二

- Source URL: https://zh.wikisource.org/wiki/紫微斗數全書
- Source type: 公共领域古籍文本（Wikisource）
- Read sections:
  - 卷一目录与开头：`太微赋`、`形性赋`、`诸星问答论`目录
  - 卷二开头：`安身命例`、`安十二宫例`、`安南北斗诸星诀`、`安文昌文曲星诀`、`安左辅右弼星诀`、`安天魁天钺诀`、`安禄存星诀`、`安禄权科忌四星变化诀`

#### 关键摘录/事实

1. 卷一 `太微赋`：
   - 强调“星之分野，各有所属”，“入庙为奇，失度为虚”，“以身命为福德之本”。
   - 产品含义：紫微解释不能只列星曜，要结合宫位、庙陷、身命、三方四正/照合语境。

2. 卷二 `安身命例`：
   - 命宫：从寅上起正月，顺数至生月，再从生月起子时逆数至生时安命。
   - 身宫：从生月起子时顺数至生时安身。
   - 闰月有特殊处理提示：闰月正月生者要在二月内起安身命。
   - 工程含义：`iztro` 排盘应与这个规则做 fixture 对照，尤其是闰月、时辰边界。

3. 卷二 `安十二宫例`：
   - 十二宫顺序：命、兄弟、妻妾、子女、财帛、疾厄、迁移、奴仆、官禄、田宅、福德、父母。
   - 文中提示“男女俱从逆转切忌莫顺去”。需要进一步核对这句与现代排盘宫位排列的差异。

4. 卷二主星/辅星安法：
   - `安南北斗诸星诀` 给出紫微、天机、太阳、武曲、天同、廉贞，以及天府系诸星的排布口诀。
   - `安禄权科忌四星变化诀` 给出生年干四化表，如“甲廉破武阳”为廉贞化禄、破军化权、武曲化科、太阳化忌。
   - 工程含义：这些应成为紫微排盘 fixture 的核心规则。
