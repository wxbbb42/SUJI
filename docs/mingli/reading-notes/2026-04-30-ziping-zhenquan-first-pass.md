## 实际读过章节与摘录/大意

### 八、论用神

短摘录：`八字用神，专求月令，以日干配月令地支，而生克不同，格局分焉。`

大意：格局法的“用神”首先不是现代平衡派里泛称的补救五行，而是以日干配月令所成的格局核心。财官印食为善，宜顺用；杀伤劫刃为不善，宜逆用。建禄/月劫一类月令“本身不可为用”时，仍须以月令为主，再看四柱有无财官杀食透干会支。

对 repo 的影响：
- `BaziEngine.determineGeJu` 当前“取月支主气十神命名”为合理的 MVP 入口，但《子平真诠》强调月令所藏、透干、会支与顺逆配合，不等于主气直出即可完成格局判定。
- 文案中应区分“格局用神（月令成格核心）”与“扶抑/调候用神”。

### 九、论用神成败救应

短摘录：`用神专寻月令，以四柱配之，必有成败。`、`成中有败，必是带忌；败中有成，全凭救应。`

大意：同一月令用神要与四柱配合，才有成格、败格、带忌、救应。官、财、印、食、杀、伤、阳刃、建禄月劫都有各自成败条件；破格后若有印解伤、合杀取清、会合解刑冲、食化劫、财去印存食等，可能“救应”。

对 repo 的影响：
- 当前 `GeJu.strength` 只用月令主气五行是否等于 `yongShen/jiShen` 粗分上中下，不能表达“成败救应”。
- 解释层不要把格局名称等同结论；至少要标 `mvp_simplified`，未来需加 `cheng/bai/jiuYing` 或 evidence 字段。

### 十、论用神变化

短摘录：`用神既主月令矣，然月令所藏不一，而用神遂有变化。`、`八字非用神不立，用神非变化不灵。`

大意：月令不是单一静态标签。除子午卯酉外，地支多有藏干；月令本主不透而其他藏干透出，或地支会合成局，都可能使命局由原本的官/伤/财/印等转化。变化有善有不善，也有“变而不失本格”。

对 repo 的影响：
- 当前 `determineGeJu` 没检查月令藏干透出优先级，也没纳入三合/三会导致的格局变化。
- `checkHuaQiGe` 只处理天干五合化气，不能代表《子平真诠》的“用神变化”。

### 十二、论用神格局高低

短摘录：`八字既有用神，必有格局，有格局必有高低。`、`其理之大纲，亦在有情、有力无力之间而已。`

大意：格局高低不由格名自动决定；财官印食杀伤劫刃皆有贵贱。关键大纲是“有情/无情、有力/无力”，并且可能“一字而有千钧之力，或半字而败全局之美”。

对 repo 的影响：
- `strength: 上/中/下` 目前不能表达有情、有力、相神是否破、喜忌是否得助等要素。
- UI/AI 输出若显示“上格/下格”应谨慎，最好改成“当前算法粗评”。

### 十五、论相神紧要

短摘录：`月令既得用神，则别位亦必有相，若君之有相，辅者是也。`、`凡全局之格，赖此一字而成者，均谓之相也。`、`伤用神甚于伤身，伤相甚于伤用。`

大意：相神是辅助月令用神、使命局成格的关键字，不是固定十神表。例如官用财生时，官为用、财为相；杀逢食制时，杀为用、食为相。但原文也说这只是定法，不是通变之妙，凡全局赖以成格之字都可为相。相神受伤，格局立败。

对 repo 的影响：
- 当前数据结构没有相神概念。若以后做格局解释，需要输出“月令用神 / 相神 / 忌神 / 救应”的关系图，而不是只输出一个格局名。

### 二十一、论星辰无关格局

短摘录：`八字格局，专以月令配四柱，至于星辰好歹，既不能为生克之用，又何以操成败之权？`、`格局既成，即使满盘孤辰入煞，何损其贵？格局既破，即使满盘天德贵人，何以为功？`

大意：神煞/星辰不是格局成败主轴；不能因为吉星就抛开用神四柱，不能因为凶煞就否定成格。部分古书所说“禄贵”等可能指正官/财官，而非天乙贵人等星煞。

对 repo 的影响：
- `shenSha` 可作为传统文化标签或辅助提示，但不能覆盖格局、生克、月令判断。
- 产品解释要避免“有天德贵人所以必贵”“孤辰寡宿所以必凶”一类强断。

### 二十五、论行运

短摘录：`论运与看命无二法也。看命以四柱干支，配月令之喜忌，而取运则又以运之干，配八字之喜忌。`、`有似喜而实忌者`、`有似忌而实喜者`。

大意：行运要把运干/运支与原局干支、月令喜忌统观。喜忌不是单字查表：同一类干支可能因合、冲、会、根、混杂而结果不同；有行干不行支、行支不行干、同类不同用、冲有缓急轻重、逢冲不冲等复杂情况。

对 repo 的影响：
- `DayunEngine` 或解释层不能只把某大运五行映射为固定吉凶。
- 行运模块应读取格局上下文、月令喜忌、相神/忌神/救应，至少在 MVP 文案中标明“简化”。

## Claims / 工程落点

本次已新增或更新 `docs/mingli/claims.json` 中与以下主题有关的 claims：

- 月令用神：格局法以月令为主，但需透干、藏干、会支与四柱配合。
- 相神：成格所赖之字为相神；相神受伤比用神受伤更关键。
- 成格破格：格名不是结论，需判成败、带忌、救应。
- 星辰不入格局：神煞/星辰不能操纵格局成败，只能作辅助文化标签。
- 行运通变：大运按原局格局喜忌统观，不能按单一五行/神煞死断。
- `BaziEngine.determineGeJu`：当前是 MVP heuristic，尤其缺少透干取格、会合变化、相神、成败救应。
