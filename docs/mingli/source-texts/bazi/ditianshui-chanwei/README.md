- **体用论** → `RiZhuStructure` 的 `deLing/shiLing/zuoGen` 字段

- **顺逆论** → 从格 / 化气格 / 专旺格的判定

## 录入注意

1. 任铁樵注分两层：**原文（4-7 字一句的歌诀）** + **任注（白话注释）+ 命例**。请用 `## 原文` / `## 任注` / `## 命例` 三段分开
2. 命例（如"某某八字"）尤其重要，是 Phase 2 fixture 的金矿——请完整录入命例的八字 + 任铁樵分析
3. 任注引用其他章节时（"详见某某章"），保留交叉引用，便于后续校注
