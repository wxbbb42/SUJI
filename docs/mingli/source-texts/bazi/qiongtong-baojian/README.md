1. **每月独立小节**：用 `## 寅月（正月）` / `## 卯月（二月）` 这种二级标题分隔，方便我按月解析
2. **关键字段需明确标出**——每月通常包含：
   - 总论（"正月甲木，初春余寒，丙火为先，癸水次之..."）

   - 喜神 / 佐神（辅助天干）
   - 忌神（避免的天干）
