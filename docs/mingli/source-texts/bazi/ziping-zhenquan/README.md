**Phase 2 算法主框架来源**。BaziEngine.ts 的 `determineGeJu` 重写完全按本书的"取格 → 成败 → 救应 → 相神 → 高低"流程做。其中：

- **抽象章节**（00-abstract-chapters.md）= `determineGeJu` 主框架
